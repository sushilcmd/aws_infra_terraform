### TERRAFORM POC
    Create AWS infrastructure using terraform

## STEPS

####  S-1
    Install terraform using given URL 
    
    https://www.terraform.io/downloads.html

#### S-2
    Clone this repo
    run given command

    terraform init
    terraform fmt
    terraform refresh
    terraform show --json
    terraform apply -auto-approve

#### S-3
    For destroying the infrastructure run given command

    terraform destroy
